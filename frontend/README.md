# 🃏 JoJo Stand Gallery (Frontend)

Interfaz de usuario interactiva para la visualización y administración de la galería de Stands.

## 🛠️ Tecnologías
*   **Framework:** React con TypeScript y Vite
*   **Gráficos:** Chart.js (Radar Charts)
*   **Estilos:** CSS3 con animaciones para prismas 3D.
*   **Cliente API:** Axios

## ✨ Funciones
*   Visualización de personajes en tarjetas tipo prisma 3D.
*   Gráficos dinámicos de estadísticas (Poder, Velocidad, etc.).
*   Formulario interactivo para registrar y editar nuevos Stands en tiempo real.
