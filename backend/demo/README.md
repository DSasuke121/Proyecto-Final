# 🗡️ JoJo Stand API (Backend)

Este es el servidor encargado de la gestión de datos de la aplicación. Proporciona una API RESTful para manejar el inventario de Stands y sus usuarios.

## 🛠️ Tecnologías
*   **Lenguaje:** Java 21
*   **Framework:** Spring Boot
*   **Base de Datos:** H2 (En memoria)
*   **Despliegue:** Docker en Render

## ✨ Funciones
*   Gestión de base de datos (CRUD).
*   Carga automática de datos iniciales (*Star Platinum*).
*   Endpoint centralizado para el consumo del frontend.
